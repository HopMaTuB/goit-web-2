CLI application "Personal Assistant":

1. all - show all commands
2. add <name> <phone> - add a new contact
3. change <name> <old_phone> <new_phone> - change phone number
4. phone <name> - find phone number
5. birthday <name> - add new birthday
6. birthdays - show upcoming birthdays
7. all_commands - show all commands
8. hello - receive a greeting from the bot
9. close or exit - close application
